# K90R
A Custom 90 Keys Keyboard including Rotary Encoder Switch, OLED powered by Rasberry Pi Pico MCU and KMK Firmware.

## Features
- Per-Key RGB LED
- Hotswap Sockets
- OLED Display
- Rotary Encoder
